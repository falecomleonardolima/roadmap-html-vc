# Curriculum Vitae - Single Page CV

## About this project

This project was developed as an exercise proposed at [https://roadmap.sh/projects/single-page-cv](https://roadmap.sh/projects/single-page-cv).

## Development

- **HTML**: Generated manually, with a focus on semantics and accessibility
- **AI used**: GitHub Copilot (Claude Haiku 4.5)
  - Function: Support for HTML code validation
  - Accessibility best practices verification
  - Suggestions for structural improvements

## Demo / Testing

You can test the project at: https://falecomleonardolima.github.io/roadmap-html-vc/
